# Rechner-Einfach-
Ein einfacher Taschenrechner ist ein Programm, das grundlegende mathematische Berechnungen durchführen kann. Der Benutzer gibt Zahlen und Rechenzeichen ein, und das Programm berechnet daraus ein Ergebnis. Unterstützt werden die Grundrechenarten Addition, Subtraktion, Multiplikation und Division.
